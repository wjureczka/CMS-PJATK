# CMS-PJATK
CMS-PJATK is a final exam study project. It consists of 2 subprojects: frontend and backend. The main purpose is to recreate Content Management System from the very bottom.

## Prerequisites
- npm version >=7.x.x
- node lts >=12.x.x

## Installation
```bash
npm i
```

## How to run in development mode
```bash
npm run cms:start
```
## How to build
```bash
npm run cms:build
```
